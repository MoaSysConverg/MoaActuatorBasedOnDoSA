# MoaActuatorBasedOnDoSA

DoSA-2D, DoSA-3D 및 PyAEDT Actuator 자동화 코드를 통합한 액추에이터 설계/해석 툴킷입니다.

## 구조

```
MoaActuatorBasedOnDoSA/
├── DoSA-3D/          # DoSA-3D 액추에이터 3D 설계 툴 (C#)
├── DoSA-2D/          # DoSA-2D 액추에이터 2D 설계 툴 (C#)
└── pyaedt_actuator/  # PyAEDT 기반 Maxwell 해석 자동화 (Python)
```

## 출처

- DoSA-3D: https://github.com/OpenActuator/DoSA-3D
- DoSA-2D: https://github.com/OpenActuator/DoSA-2D
- pyaedt_actuator: https://github.com/enjoyneer87/pyAEDT_MoaEM
